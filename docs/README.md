# Documentação WAFrame

Bem-vindo à documentação do WAFrame, um framework PHP para desenvolvimento rápido de aplicações.

## Documentação Disponível

### [Image - Processamento de Imagens](Image.md)
Classe completa para processamento de imagens com suporte a redimensionamento, recorte, conversão de formatos e criação de thumbnails.

**Recursos:**
- Suporte a GIF, JPEG, PNG e WebP
- Redimensionamento com manutenção de proporção
- Recorte de imagens
- Criação automática de thumbnails
- Validação de segurança (MIME types, magic bytes)
- Tratamento de erros com exceptions customizadas
